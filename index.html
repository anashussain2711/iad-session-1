(Project Overview)

Mastermind:
Race Against Time

We began by presenting the concept of our game, Mastermind: Race Against Time. The game challenges players to crack a secret code in a tense bomb-defusal scenario. Our demonstration showcased the game's UI and functionality, followed by an explanation of the underlying code (HTML, CSS, and JavaScript).

(Journey Through the Game UI)

A walkthrough of our game’s key screens.

Our game features a clean and interactive UI that enhances the player experience. Below, we highlight the three main screens of the game, each designed with simplicity and functionality in mind.

Title Screen

The title screen welcomes players with a bold and dynamic design. It features the game name, a brief tagline, and a clear call-to-action to start or explore the rules.

Gameplay Screen

The gameplay screen is where the action unfolds. Players interact with the bomb interface to enter their guesses, guided by a responsive design and visual feedback.

End Screen

The conclusion screen delivers a personalized message of victory or defeat. It diusplays Message congratulating the player or encouraging another attempt.
